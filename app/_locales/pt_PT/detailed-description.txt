Esta extensão agrupa os canais do Slack por prefixo, tornando mais fácil encontrar os canais visualmente.

🛠️ Recursos:
- Agrupa automaticamente canais com prefixos comuns
- Suporta tanto hífenes (-) como sublinhados (_)
- Funciona com nomes de canais como:
  • prefixo-nome
  • prefixo_nome

🚀 Como usar:
- Simplesmente instale a extensão
- Abra o Slack no seu navegador
- Os canais serão automaticamente agrupados por prefixo

🔒 Privacidade:
- Sem recolha de dados
- Sem rastreamento
- Nenhuma informação do Slack é transmitida a desenvolvedores ou terceiros

💻 O código fonte também está disponível, para que possa compilar e instalar você mesmo.
Para mais informações, por favor, visite:
