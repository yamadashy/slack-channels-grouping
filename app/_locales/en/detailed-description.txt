This extension groups Slack channels by prefix, making it easier to find channels visually.

🛠️ Features:
- Groups channels with common prefixes automatically
- Supports both hyphens (-) and underscores (_) for grouping
- Works with channel names like:
  • prefix-name
  • prefix_name

🚀 How to Use:
- Simply install the extension
- Open Slack in your browser
- Channels will be automatically grouped by prefix

🔒 Privacy:
- No data collection
- No tracking
- No information from Slack is transmitted to developers or third parties

💻 The source code is also available, and you can build and install it yourself.
For more details, please visit:
https://github.com/yamadashy/slack-channels-grouping
