# Datenschutzrichtlinie

Slack Channels Grouping sammelt keine Daten.

In Zukunft könnte Slack Channels Grouping die Möglichkeit hinzufügen, Benutzereinstellungen auf Ihrem Gerät zu speichern, aber diese Daten werden nicht an den Server gesendet.

Darüber hinaus wird das Slack Channels Grouping-Verwaltungskonto nicht an Dritte übertragen.

Bei Fragen oder Anfragen zur Datenschutzrichtlinie wenden Sie sich bitte an koukun0120@gmail.com.
