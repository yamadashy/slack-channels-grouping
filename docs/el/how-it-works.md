# Πώς Λειτουργεί

Το Slack Channels Grouping οργανώνει αυτόματα την πλευρική γραμμή του Slack σας ομαδοποιώντας τα κανάλια που μοιράζονται το ίδιο πρόθεμα.

## Ανίχνευση Προθέματος

Η επέκταση ανιχνεύει τα προθέματα αναζητώντας την πρώτη `-` (παύλα) ή `_` (κάτω παύλα) στο όνομα του καναλιού.

| Όνομα Καναλιού | Ανιχνευμένο Πρόθεμα |
|----------------|---------------------|
| `dev-api` | `dev` |
| `dev-frontend` | `dev` |
| `project_backend` | `project` |
| `team-design-v2` | `team` |
| `general` | (χωρίς πρόθεμα) |

## Παράδειγμα Ομαδοποίησης

Τα κανάλια με το ίδιο πρόθεμα ομαδοποιούνται οπτικά μαζί:

**Πριν:**
```
# dev-api
# dev-frontend
# dev-backend
# sales-leads
# sales-reports
# general
```

**Μετά:**
```
# ┬ dev-api
# ├ dev-frontend
# └ dev-backend
# ┬ sales-leads
# └ sales-reports
# general
```

## Οπτικοί Δείκτες

Η επέκταση χρησιμοποιεί διαχωριστικά τύπου δέντρου για να δείξει την ομαδοποίηση:

| Σύμβολο | Σημασία |
|---------|---------|
| `┬` | Πρώτο κανάλι σε μια ομάδα |
| `├` | Μεσαίο κανάλι σε μια ομάδα |
| `└` | Τελευταίο κανάλι σε μια ομάδα |

## Υποστηριζόμενοι Τύποι Καναλιών

| Τύπος Καναλιού | Ομαδοποιημένο |
|----------------|---------------|
| Δημόσια κανάλια | Ναι |
| Ιδιωτικά κανάλια | Ναι |
| Άμεσα μηνύματα | Όχι |
| Ομαδικά DM | Όχι |

Τα άμεσα μηνύματα και τα ομαδικά DM εξαιρούνται σκόπιμα από την ομαδοποίηση για να διατηρούνται οι συνομιλίες σας εύκολα προσβάσιμες.

## Μηδενική Διαμόρφωση

Απλά εγκαταστήστε την επέκταση και λειτουργεί αυτόματα. Δεν απαιτείται ρύθμιση ή διαμόρφωση.
