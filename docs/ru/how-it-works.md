# Как это работает

Slack Channels Grouping автоматически организует вашу боковую панель Slack, группируя каналы с одинаковым префиксом.

## Определение префикса

Расширение определяет префиксы, находя первый `-` (дефис) или `_` (подчёркивание) в названии канала.

| Название канала | Определённый префикс |
|-----------------|----------------------|
| `dev-api` | `dev` |
| `dev-frontend` | `dev` |
| `project_backend` | `project` |
| `team-design-v2` | `team` |
| `general` | (без префикса) |

## Пример группировки

Каналы с одинаковым префиксом визуально группируются вместе:

**До:**
```
# dev-api
# dev-frontend
# dev-backend
# sales-leads
# sales-reports
# general
```

**После:**
```
# ┬ dev-api
# ├ dev-frontend
# └ dev-backend
# ┬ sales-leads
# └ sales-reports
# general
```

## Визуальные индикаторы

Расширение использует разделители в стиле дерева для отображения группировки:

| Символ | Значение |
|--------|----------|
| `┬` | Первый канал в группе |
| `├` | Средний канал в группе |
| `└` | Последний канал в группе |

## Поддерживаемые типы каналов

| Тип канала | Группируется |
|------------|--------------|
| Публичные каналы | Да |
| Приватные каналы | Да |
| Личные сообщения | Нет |
| Групповые DM | Нет |

Личные сообщения и групповые DM намеренно исключены из группировки, чтобы ваши разговоры оставались легко доступными.

## Без настройки

Просто установите расширение, и оно работает автоматически. Никакой настройки не требуется.
