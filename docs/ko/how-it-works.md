# 작동 방식

Slack Channels Grouping은 동일한 접두사를 공유하는 채널을 그룹화하여 Slack 사이드바를 자동으로 정리합니다.

## 접두사 감지

확장 프로그램은 채널 이름에서 첫 번째 `-` (대시) 또는 `_` (밑줄)을 찾아 접두사를 감지합니다.

| 채널 이름 | 감지된 접두사 |
|-----------|---------------|
| `dev-api` | `dev` |
| `dev-frontend` | `dev` |
| `project_backend` | `project` |
| `team-design-v2` | `team` |
| `general` | (접두사 없음) |

## 그룹화 예시

동일한 접두사를 가진 채널은 시각적으로 함께 그룹화됩니다:

**적용 전:**
```
# dev-api
# dev-frontend
# dev-backend
# sales-leads
# sales-reports
# general
```

**적용 후:**
```
# ┬ dev-api
# ├ dev-frontend
# └ dev-backend
# ┬ sales-leads
# └ sales-reports
# general
```

## 시각적 표시

확장 프로그램은 그룹화를 표시하기 위해 트리 스타일 구분자를 사용합니다:

| 기호 | 의미 |
|------|------|
| `┬` | 그룹의 첫 번째 채널 |
| `├` | 그룹의 중간 채널 |
| `└` | 그룹의 마지막 채널 |

## 지원되는 채널 유형

| 채널 유형 | 그룹화 |
|-----------|--------|
| 공개 채널 | 예 |
| 비공개 채널 | 예 |
| 다이렉트 메시지 | 아니오 |
| 그룹 DM | 아니오 |

다이렉트 메시지와 그룹 DM은 대화에 쉽게 접근할 수 있도록 의도적으로 그룹화에서 제외됩니다.

## 설정 불필요

확장 프로그램을 설치하면 자동으로 작동합니다. 설정이나 구성이 필요 없습니다.
